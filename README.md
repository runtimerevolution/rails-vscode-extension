# Contributing

### Installing locally
First make sure you have everything installed by running:
```
yarn install
```
Then package the extension by running:
```
yarn vsce package
```
And finally install the generated .vsix file:
```
code --install-extension NAME_OF_THE_VSIXFILE
```
